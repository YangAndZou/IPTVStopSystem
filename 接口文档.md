#						接口#

##查看日志##

url：/logs

接收参数：无

返回值：模板对象 program_logs，内含：

```
频道名：{{ program_logs.program.program_name }}

操作内容：{{ program_logs.content }}

操作时间：{{ program_logs.update_time }}
```



## 频道的关停 / 恢复  

url: /

接收参数：

```
"mode": "turn_off" / "turn_on"

无论关停还是恢复都要的
"program_ips": [] 一个ip列表
```

返回值：

```
成功时：
'msg': 'ok', 'code': ‘200’

失败时：
'msg': e, 'code': "201"
```
